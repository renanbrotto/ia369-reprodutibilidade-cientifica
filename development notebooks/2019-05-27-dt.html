Introdução¶

As tcnicas de Aprendizado de Mquina esto em rpida ascenso em diversas reas, como processamento de linguagem natural e reconhecimento de fala \cite{Kamath2019}, processamento de imagem \cite{Cipolla2013}, problemas de classificao \cite{Duda2000}, reconhecimento de padres \cite{Bishop2006} entre outras.

O objetivo central do Aprendizado de Mquina projetar modelos computacionais capazes de aprender uma determinada tarefa a partir de dados, sem que exista uma programao especfica para a tarefa em questo \cite{Samuel1959}. Para tanto, apresentamos um conjunto de dados disponveis, denominado conjunto de treinamento, a fim de que o modelo capture as informaes relevantes para o problema investigado \cite{Hastie2009}.

Alm de ter um bom desempenho, segundo uma mtrica adequada, frente aos dados de treinamento, o modelo ajustado deve ser capaz de apresentar um bom desempenho frente a dados desconhecidos, pertencentes a um conjunto normalmente denominado conjunto de teste. Este segundo requisito configura a capacidade de generalizao do modelo \cite{Bishop2006}, \cite{haykin-rn}.

Contudo, durante o processo de aprendizagem, o modelo pode capturar informaes irrelevantes, ou mesmo indesejadas, para a tarefa considerada, configurando uma situao de sobreajuste \cite{haykin-rn}. Neste tipo de situao, temos um bom desempenho frente ao conjunto de treinamento, mas o modelo se comporta mal quando avaliado frente a novos dados \cite{Bishop2006}. Dentre as informaes indesejadas aprendidas, podemos citar rudo presente sobre os dados \cite{Bishop2006} ou mesmo algum tipo de vis apresentado nos dados de treinamento.

O vis contido nos dados de treinamento desempenha um papel fundamental quando aplicamos os modelos de Aprendizado de Mquina em tarefas com impacto social direto, tais como concesso de crdito \cite{ONeil2016}., aprovao em universidades \cite{ONeil2016} ou a concesso de liberdade condicional \cite{Chouldechova2017}. Neste tipo de problema as tcnicas clssicas de aprendizado incorporam as tendncias observadas nos dados, o que pode dar origem a atributos discriminativos, como por exemplo o gnero em um problema de concesso de crdito ou a etnia em um problema de concesso de liberdade condicional. Uma vez aprendida esta caracterstica, o modelo ajustado passa a produzir resultados que podem contribuir ainda mais com a disparidade do problema \cite{ONeil2016}.

Objetivo¶

Nosso objetivo neste trabalho aplicar as tcnicas de ICA \cite{Hyvarinen2001} sobre os atributos do nosso problema de modo a promover a equidade na classificao. Para tanto, vamos investigar a descorrelao linear, tendo como base o trabalho \cite{Zafar2017}, e a descorrelao no-linear, que uma condio mais prxima da independncia.

Metodologia¶

Para a tarefa de classificao, empregaremos um classificador ajustado segundo o paradigma de regresso logstica. Para os nossos propsitos aqui, adotaremos o atributo "Gnero (Sex)" como atributo discriminatrio. Ao todo, empregaremos 3 classificadores:

1) Classificador Tradicional: neste caso, faremos o treinamento sem nenhum tipo de ajuste sobre os dados.

2) Classificador com Descorrelao Linear: neste segundo caso, buscaremos descorrelacionar linearmente o atributo discriminatrio das classes do problema ("Fairness Constraints: Mechanisms for Fair Classification", Zafar, et al., 2017).

3) Classificador com Descorrelao No Linear: neste ltimo caso, buscaremos a independncia entre o atributo Gnero e as classe do problema, por meio da descorrelao no-linear.

Conclusões¶

Neste trabalho avaliamos como algumas das tcnicas de ICA, i.e, descorrelao linear e descorrelao no-linear, podem ser usadas para promover a equidade na tarefa de classificao. Para o caso particular que estudamos, ambas as tcnicas levaram a uma reduo de acurcia nos classificadores. Contudo, a descorrelao linear levou a uma pequena aproximao entre as populaes masculina e feminina, em termos de indivduos classificados como "Alta Renda". No observamos a mesma aproximao para quando empregamos a descorrelao no-linear.
